# Quantitative analysis of a Student's t random walk

## Overview

This project explores how **heavy-tailed distributions** affect the statistical behavior of financial returns. Instead of using the classical Gaussian random walk, we simulate **random walks with shocks drawn from a Student’s t-distribution** and analyze the resulting properties.

The goal is to demonstrate how **extreme events (outliers)** can naturally emerge from heavy-tailed distributions without requiring temporal correlations, trends, or volatility clustering.

Through simulation and statistical analysis, the project illustrates several well-known empirical features of financial markets, particularly the presence of **fat tails** in return distributions.

## Objectives

The main objectives of this project are:

* Construct a **t-Student random walk** model for asset prices
* Analyze the **distribution of returns** generated by the model
* Detect and visualize **heavy tails**
* Compare the distribution with a **Gaussian benchmark**
* Test for **power-law behavior** in the tails
* Verify the absence of **autocorrelation** and **volatility clustering**

## Tools and Libraries
The analysis was conducted using Python with the following libraries:  
- **pandas**: for time series manipulation.  
- **numpy**: for numerical computations and logarithmic transformations.  
- **matplotlib**: for visualizations of distributions and time series.  
- **scipy** and **statsmodels**: for statistical tests, regressions, and autocorrelation analysis.

## Key Findings

The simulations reveal several important results:

* The return distribution exhibits **high kurtosis**, indicating heavy tails.
* Extreme values occur **far more frequently** than in a Gaussian model.
* QQ plots show clear **deviations from normality** when the degrees of freedom are low.
* The tails follow a **power-law distribution** with exponent ( \alpha ) between **2 and 4**.
* No significant **autocorrelation** or **volatility clustering** is detected.

These results demonstrate that **heavy-tailed distributions alone can generate extreme market events**, even in the absence of temporal correlations.
